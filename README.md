#ZssVimConfig
本vim配置采用vundle管理插件配置，这样可以仅仅保留一份.vimrc即可。
在本工程目录向运行
```
    bash install.sh
```
运行完上述指令后，vundle 插件会安装完成,打开vim后，在命令模式下输入如下指令
```
    :BundleInstall
```
等待一段时间后，可以安装这些插件。



-   自动补全
-   高亮显示
-   书签以及书签跳转
-   SQL关键字小些改为大写
-   连接数据库dbext操作
-   markdown相关
-   C/C++/Golang/Scala/python等继承IDE
